# 🗓️ Calendar Planner Pro

A modern, high-performance Flutter application designed for seamless daily scheduling. This app features a beautiful Material 3 interface, local authentication, and a real-time background notification engine to keep you organized.

---

## ✨ Features

- **🔐 Secure Local Auth**: Integrated Login and Registration system powered by `shared_preferences`.
- **📅 Interactive Custom Calendar**: Navigate through any month or year with a custom-built grid interface.
- **🌗 Dynamic Theming**: Seamlessly switch between **Day Mode** (Light) and **Night Mode** (Dark) with state persistence.
- **🔔 Real-time Notifications**: Background polling engine that triggers in-app alert banners at the exact time of your scheduled events.
- **📑 Detailed Event Management**: Add tasks with titles, specific times, and personalized notes.
- **🔍 Smart Filtering**: Quickly toggle between **Upcoming**, **Past**, and **All** events.
- **⚡ Smooth UX**: Animated page transitions and staggered list animations for a premium feel.
- **📱 Responsive Design**: Optimized for both mobile and tablet screen ratios.
